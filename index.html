Habilidades

HTML5

HTML é uma linguagem de marcação para web, com suas inúmeras tags, para informar os navegadores como exibir conteúdo. Sempre manter as tags o máximo de semântica possível.

CSS3

CSS mais conhecida como folhas de estilo em cascata, é a linguagem de marcação para o design da web, para estilizar as páginas na web e fazer layouts incríveis.

JAVASCRIPT

Java Script uma linguagem de programação, também focada para web, para deixar as páginas mais dinâmicas, e assim dando vida para as páginas web.

TYPESCRIPT

TypeScript uma linguagem que foi criada pela Microsoft em cima do JavaScript, para adicionar recursos de tipagem estáticas à linguagem, para melhora lá, e também é uma linguagem orientada a objetos.

NODE JS

No momento estou estudando Node Js, mas eu vejo que node js é um ambiente de execução para usar o Java Script do lado do servidor. Node js executa programação assíncrona de thread único.

BOOTSTRAP 5

Bootstrap é uma biblioteca, para facilitar a vida do programador, com ela você consegue fazer muitas páginas dinâmicas com pouca linha de código HTML, e CSS e assim você se preocupa mais com o back-end.
