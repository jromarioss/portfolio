Habilidades

HTML5

HTML é uma linguagem de marcação para web, com suas inúmeras tags, para informar os navegadores como exibir conteúdo. Sempre manter as tags o máximo de semântica possível.

CSS3

CSS mais conhecida como folhas de estilo em cascata, é a linguagem de marcação para o design da web, para estilizar as páginas na web e fazer layouts incríveis.

JAVASCRIPT

Java Script uma linguagem de programação, também focada para web, para deixar as páginas mais dinâmica, e assim dando vida para as páginas web.

TYPESCRIPT

TypeScript uma linguagem que foi criada pela Microsoft em cima do JavaScript, para adicionar recursos de tipagem estáticas à linguagem, para melhora, e também uma linguagem orienda a objetos.

NODE JS

No momento estou estudando Node Js, mais o node js é um ambiente de execução para usar o Java Script do lado do servidor.O Node js executa programação assíncrona de thread único.
